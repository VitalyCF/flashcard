# Flashcard

Spaced repetition algorithm.

This web app uses several modern web technologies:

* Web components, Polymer Starter Kit
* Installable Web App
* Lovefield Query engine
* IndexedDB
* 3D CSS animation
* Service Worker
* Push Notification
* Google App Engine



## Deploy on appengine

appcfg.py -A flashcard-121 update flashcard/
